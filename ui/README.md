# Your Package Name

A customized React component library built with TypeScript and Tailwind CSS.

## Installation

```bash
npm install @glint-ui/react
# or
yarn add @glint-ui/react
# or
pnpm add @glint-ui/react
```

## Usage

```tsx
import { BaseButton, BaseCard } from '@glint-ui/react';

function App() {
  return (
    <BaseCard>
      <BaseButton>Click me</BaseButton>
    </BaseCard>
  );
}
```

## Components

This library includes the following components:

- **Base Components**: Button, Card, Avatar, Input, etc.
- **Form Components**: Checkbox, Radio, Select, etc.
- **Layout Components**: Accordion, Tabs, Pagination, etc.

## Requirements

- React >= 18
- Next.js >= 13
- Tailwind CSS
